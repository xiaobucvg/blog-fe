1. webpack    -- 打包工具

3. bootstrap  -- UI
4. juicer     -- 模板渲染

3. 侧边栏
    - 博客导航

2. 首页

#### 2020/3/14

文章详情页开发笔记：

**文章排版**
使用了 typo.css . 将文章的父容器添加类 .typo 即可将文章内容进行排版。

**响应式表格**
使用 bootstarp 自带的 .table-responsive 将 .taable 包裹即可.

**代码高亮**
使用了库 Prism ，引入核心 css 与主题 css 即可. 需要将代码部分做处理，可以在接收到后端文章数据后对代码部分进行处理并替换。

**自动生成目录**
使用了 jquery 插件 autoMenu，并做了一些小的改动，使其支持二级标题缩进等.
